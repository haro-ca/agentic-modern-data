02 Datos

Módulo 02 — Polars & DuckDB

Polars — transformación Python-nativa
polars_demo.py
import polars as pl df = pl.read_csv("sales_transactions.csv") # Lazy evaluation — eficiente df.lazy() .filter(pl.col("totalPrice") > 20) .group_by("franchiseID") .agg(pl.col("totalPrice").sum()) .sort("totalPrice", descending=True) .collect()
🤔 Escribo mucho Python para lo que SQL hace en una línea...
DuckDB — SQL sobre cualquier cosa
duckdb_demo.py
import duckdb # SQL directo sobre CSV (sin cargar) duckdb.sql(""" SELECT franchiseID, SUM(totalPrice) AS revenue FROM 'sales_transactions.csv' GROUP BY 1 ORDER BY 2 DESC """) # 🤯 Attach directo a Postgres duckdb.sql(""" ATTACH 'postgresql://...' AS neon (TYPE postgres) """)
🤩 Sin ETL. DuckDB consulta Postgres directamente desde Python.

03 Databricks

Módulo 03 — Databricks

Subir datos a Delta Lake y construir un Genie space (lenguaje natural → SQL).

CLI + catálogo
terminal
# Configurar acceso databricks configure # Ver catálogos y tablas databricks catalogs list databricks schemas list samples databricks tables list samples bakehouse # Descargar tabla como CSV databricks api post \ /api/2.0/sql/statements \ --json '{"statement":"SELECT * FROM samples.bakehouse .sales_customers", "warehouse_id":"..."}'
Genie space
  • Seleccionar tablas del catálogo Unity
  • Crear un Genie space desde la UI
  • Añadir contexto de negocio
  • Preguntar en lenguaje natural
🤩 "¿Cuál fue la franquicia con más ventas en junio?" — Genie genera el SQL solo.
